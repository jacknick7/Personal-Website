Building a website

Projects | Published June 2020 | 5 min read

Since I never learned how to build a website in college, I decided to create one on my own. My goal was to make a really simple static web using HTML and CSS and learn the basics of both languages in the process. If you are reading this you have probably already seen the end result of this project.

At first, I watched a short tutorial about HTML and CSS in which I made a toy website. Once I felt that my knowledge of those languages was good enough, I started looking at personal websites templates to get ideas for my own web (my graphic design skills are non-existent). In the end, I built a website with two types of pages: index and entry. I am happy with the final result, even if I feel like there is a lot of complex knowledge about HTML/CSS that I still lack, because I made the website I wanted.

In order to upload my website to the Internet I used one of the most famous hosting providers. I also downloaded the well-known FTP solution FileZilla so that I could manage my web directories with ease.
