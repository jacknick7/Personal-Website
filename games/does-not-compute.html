Does Not Compute

Games | Published July 2020 | 5 min read

The game contains three levels with ten routes in each. The player must complete them in order to win the game. To do this, you will need to complete all routes in a given time. Initially, it will start with a car and will change in order to complete all the routes. Not only that, but you also need to avoid previous movements on previous routes at the same level. If the user overturns or makes a mistake, they will receive a time penalty and will have to start the route again. In the event of a collision, the vehicle will appear with smoke and its speed will be significantly reduced. In each scenario you will find objects that increase your remaining time. To pass the level you only need to complete all the routes.
